# embedz
Easy, dependency free embeds for Svelte and Vue.

```svelte
<script>
  // Supports only Svelte for now, requires Svelte 5 and above
  import { YouTube, Vimeo } from "@embedz/test";
</script>

<YouTube 
  id="KRVnaN29GvM" 
  posterquality="low"
/>
```
